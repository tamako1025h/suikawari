## Tkinterでスイカ割りのゲームを作りました。
pythonとgithubにアップロードする勉強用に作りました<br>
## 遊び方
1.pipでインストール
```
pip install git+https://github.com/tamako1025h/suikawari
```
2.動かす
```
suikawari
```
3.アンインストール
```
pip uninstall suikawari
```
